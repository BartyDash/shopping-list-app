![](./src/og.png)

## Run Locally

1. Clone the repository to your local machine:

    ```bash
    git clone https://github.com/BartyDash/hydrapp.git
    ```
2. Go to the project directory

3. Install dependencies
    ```bash
    npm install
    ```

4. Start the server
    ```bash
    npm run start
    ```
